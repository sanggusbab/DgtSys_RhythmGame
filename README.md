### Register Reference
    x31: Time wire
    x29: Interrupt wire
    x30, x28: status register
    x27:1000
    x26: (melody frequency address)
    x25: (melody time address)
    x24: 2^19
    x23: 2^12
    x22: (DataMemory Address)
    x2: WindowStatus
    x0: hardwired zero
